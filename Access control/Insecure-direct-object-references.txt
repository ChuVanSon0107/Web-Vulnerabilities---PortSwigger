Lab: Insecure direct object references

Vulnerability: This lab stores user chat logs directly on the server's file system, and retrieves them using static URLs.

Goal: Solve the lab by finding the password for the user carlos, and logging into their account.

Analysis and Solution:
1. Click Live chat and chat something
=> Then, click View transcript
=> A request is sent: GET /download-transcript/2.txt

2. Using Burp Intruder to make a lot of request GET /download-transcript/{number}.txt

    HTTP/2 200 OK
    Content-Type: text/plain; charset=utf-8
    Content-Disposition: attachment; filename="1.txt"
    X-Frame-Options: SAMEORIGIN
    Content-Length: 520
