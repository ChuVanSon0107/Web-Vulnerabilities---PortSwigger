Lab #3 - DOM XSS in document.write sink using source location.search

Vulnerability - This lab contains a DOM-based cross-site scripting vulnerability in the search query tracking functionality. It uses the JavaScript document.write function, which writes data out to the page. The document.write function is called with data from location.search, which you can control using the website URL.

Goal - To solve this lab, perform a cross-site scripting attack that calls the alert function.
